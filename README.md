# ResumeMaker

## Overview

ResumeMaker is a web-based resume builder that enables users to create professional resumes through an intuitive interface. The application allows users to enter personal information, education, skills, projects, work experience, and certifications, generating a clean and well-structured resume suitable for internships, placements, and professional job applications.

The project focuses on providing a fast, responsive, and user-friendly experience while simplifying the resume creation process.

---

## Features

- Professional resume generation
- User-friendly interface
- Personal information management
- Education section
- Skills section
- Work experience
- Projects section
- Certifications and achievements
- Responsive design
- Resume preview

## Installation

Clone the repository:

```bash
git clone https://github.com/SakshiKShetty/ResumeMaker.git
```

Move into the project directory:

```bash
cd ResumeMaker
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

---

## Usage

1. Launch the application.
2. Enter your personal information.
3. Add education, skills, experience, projects, and certifications.
4. Preview the generated resume.
5. Download or print the resume.
